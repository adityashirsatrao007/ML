# Machine Learning Projects

This repository contains a collection of Machine Learning projects developed under the guidance of **Prof. N.B. Aherwadi**. The projects demonstrate hands-on experience with various ML algorithms, data processing techniques, and visualization methods.

## 📂 Project Structure

- `Matplotlib.ipynb` – Data visualization using Matplotlib.  
- `Numpy.ipynb` – Numerical computations and array operations using NumPy.  
- `Pandas.ipynb` – Data manipulation and analysis using Pandas.  
- `Scikit-Learn.ipynb` – Implementation of machine learning models using Scikit-Learn.  
- `Seaborn.ipynb` – Statistical data visualization using Seaborn.  
- `SIH_PS_2024.xlsx` – Dataset used in the projects.  
- `Sample-SQL-File-10-Rows.sql` – Sample SQL queries and database operations.  
- `iris.json` – Iris dataset in JSON format.  
- `file.csv` – Sample CSV file for data analysis.  
- `file.txt` – Sample text file for data processing.  
- `sample.db` – Sample SQLite database file.  
- `table-download.html` – HTML file for table download.

## 📘 About the Projects

These projects cover:

- Data preprocessing and cleaning  
- Exploratory Data Analysis (EDA)  
- Machine Learning model training and evaluation  
- Data visualization techniques  

Each notebook demonstrates practical applications of Machine Learning concepts and provides hands-on experience with real datasets.
